# Pizzeria Application

This is a full-stack pizza ordering application consisting of a React frontend and a Node.js backend.

## Project Structure

- **pizzeriaapp**: The frontend client application built with React and Bootstrap.
- **pizzariaserver**: The backend server application handling API requests.

## Prerequisites

- Node.js and npm installed.

## Installation and Setup

### 1. Backend (pizzariaserver)

Navigate to the server directory and install dependencies:

```bash
cd pizzariaserver
npm install
```

Start the server (runs on port 4000 by default):

```bash
npm start
```

### 2. Frontend (pizzeriaapp)

Open a new terminal, navigate to the app directory and install dependencies:

```bash
cd pizzeriaapp
npm install
```

Start the client application:

```bash
npm run dev
# or 'npm start' depending on your script setup
```

## Features

- **Order Pizza**: Browse a catalog of pizzas.
- **Build Your Pizza**: Custom pizza builder with ingredient selection.
- **Shopping Cart**: Manage your order items and view total cost.
